# MindVerse
MindVerse - AI-Powered Emotion & Guidance System (HTML, CSS, JavaScript)
• Created an intelligent emotion detection system using NLP & Machine Learning that understands user text/speech and identifies emotions with high accuracy.
• Designed a unique recommendation engine that maps user emotions to Bhagavad Gita shlokas, providing meaningful and personalized guidance.
• Built a structured knowledge base of verses and interpretations, enabling AI-driven emotional support and real-time suggestions.
• Developed a smooth, interactive interface that delivers an engaging user experience with clear explanations and motivational insights.
